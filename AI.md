# AI.md

## Ferramenta Utilizada

Durante o desenvolvimento do projeto foi utilizada a ferramenta ChatGPT (OpenAI) como apoio técnico e educacional.

## Objetivo da Utilização

A Inteligência Artificial foi utilizada para auxiliar no planejamento, implementação e refinamento de funcionalidades do projeto EcoSafe, desenvolvido para a disciplina de Web Development da Global Solution FIAP.

A utilização ocorreu principalmente para:

* Esclarecimento de dúvidas sobre HTML, CSS e JavaScript;
* Sugestões de organização da estrutura de pastas do projeto;
* Orientação sobre boas práticas de desenvolvimento web;
* Auxílio na implementação de funcionalidades interativas;
* Correção e depuração de erros encontrados durante o desenvolvimento.

## Aplicações no Projeto

A IA contribuiu com sugestões técnicas relacionadas a:

### Estrutura do Site

* Organização dos arquivos em diretórios específicos para CSS, JavaScript e imagens;
* Construção da estrutura semântica da página utilizando HTML5.

### Interface e Experiência do Usuário

* Desenvolvimento do layout responsivo para dispositivos móveis;
* Criação do sistema de temas (Eco, Dark e Ocean);
* Melhorias visuais no cabeçalho, cards e rodapé;
* Implementação do slideshow de imagens.

### Funcionalidades JavaScript

* Desenvolvimento do quiz interativo contendo 10 perguntas;
* Implementação do sistema de reinício do quiz;
* Validação do formulário de cadastro de alertas;
* Persistência das preferências de tema utilizando LocalStorage.

### Git e GitHub

* Orientações sobre criação e utilização de branches;
* Estratégia de commits durante o desenvolvimento;
* Configuração do GitHub Pages para publicação do projeto;
* Organização do repositório para atender aos requisitos da disciplina.

## Participação da Equipe

Todas as sugestões fornecidas pela Inteligência Artificial foram analisadas, adaptadas e implementadas manualmente pelos integrantes da equipe.

A ferramenta foi utilizada como apoio ao desenvolvimento e aprendizado, sendo as decisões finais de implementação realizadas pelos alunos.

## Considerações Finais

O uso da IA permitiu acelerar a resolução de dúvidas técnicas e facilitar a aprendizagem de conceitos relacionados ao desenvolvimento web, contribuindo para a construção do projeto sem substituir a participação ativa dos integrantes.
