# EcoSafe

## Sobre o Projeto

EcoSafe é uma solução desenvolvida para a Global Solution FIAP 2026 com o objetivo de auxiliar na prevenção de desastres naturais através do monitoramento ambiental utilizando tecnologias espaciais, satélites e inteligência artificial.

O sistema apresenta informações sobre o problema, a solução proposta, benefícios da tecnologia, além de recursos interativos para o usuário.

## Funcionalidades

- Slideshow com imagens relacionadas ao projeto
- Quiz interativo com 10 perguntas
- Formulário com validação de dados
- Sistema de temas (Eco, Dark e Ocean)
- Responsividade para dispositivos móveis
- Persistência de preferências utilizando LocalStorage
